## Trabajo Práctico 3 (Frontend)

**Objetivo:**
Desarrollar un sitio web estático utilizando HTML y CSS con estructura semántica, navegación entre páginas y formularios.

## Descripción del proyecto
Este repositorio contiene el frontend de Gimnasio Fit, un sitio web estático de presentación de servicios deportivos. Incluye múltiples páginas conectadas con navegación y formularios básicos, siguiendo los requisitos del Trabajo Práctico 3.

## Estructura de archivos
- `index.html` → Página principal / Home
- `servicios.html` → Página de servicios ofrecidos
- `equipo.html` → Página del equipo de trabajo
- `pedido.html` → Formulario de solicitud de servicios
- `contacto.html` → Formulario de contacto
- `faq.html` → Preguntas frecuentes
- `css/` → Estilos globales del sitio
  - `style.css`
  - `login.css`
- `js/` → Scripts de interacción del frontend
  - `main.js`
  - `login.js`
  - `perfil.js`
  - `register.js`
- `assets/` → Recursos estáticos (imágenes, favicon, etc.)
- `pages/` → Plantillas y páginas adicionales del sitio

## Requisitos cumplidos
- Uso de etiquetas semánticas: `header`, `nav`, `main`, `footer`
- Navegación entre páginas con rutas relativas
- CSS externo independiente para estilos
- Uso de box model (`margin`, `padding`, `border`)
- Uso de clases e IDs para diseño y estructura
- Inclusión de al menos 3 imágenes en el sitio
- Estructura de carpetas ordenada y lógica

## Cómo usar
1. Abrir `index.html` en un navegador web.
2. Navegar entre páginas usando el menú principal.
3. Completar los formularios en `pedido.html` y `contacto.html`.

## Notas importantes
- Este repositorio contiene solo el frontend del proyecto.
- El backend se encuentra en otro repositorio separado.
- El contenido está pensado como demostración de HTML, CSS y navegación estática.
